Human-Agent Development Framework

A practical framework for reducing avoidable implementation loops in agent-assisted software development.

HADF helps teams make decisions, assumptions, evidence, uncertainty, and ownership visible before they become code.

Core Idea

Every decision left open by a plan becomes a decision the agent may make during implementation. HADF improves delivery quality by making the right decisions explicit at the right time, with the right owner.

What Problem Does It Solve?

AI coding agents can produce code quickly, but the expensive part of agent-assisted development is often convergence: the work required to move from an initial request to an accepted, maintainable implementation.

For Humans

HADF keeps intent, judgement, accountability, and understanding visible throughout delivery.

For Agents

HADF gives coding agents clearer specifications, guardrails, stop rules, and validation expectations.
